Adds Recent Posts CPT for fusion builder.

== Description ==

<p>
	There is "add recent posts" element in Fusion Builder. But it works just with "post" post type. If you've created a custom post type and want to display it using page builder, you can use the "Recent Posts CPT Fusion Builder Addon".
</p>

<p>
	This plugin (Add on) will add the new element to your page builder collection. It helps you to achieve your goal.
</p>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Upload 'Recent Posts CPT Fusion Builder Addon'
3. Activate the plugin from your Plugins page.
4. Turn on Add on via the admin dashboard (Fusion Builder -> Settings -> Fusion Builder Elements).
